Python: Working with files

  • Article ID: 4099
  • Last updated on November 3, 2014
  • Authored by: Christopher Mendoza

In Python, you can open files in different modes depending on how you want to interact with the file. See the following table for an explanation of these modes.

Modes for opening files in Python

Modes Permissions
r - Read only The file pointer starts at the beginning of the file.
rb - Read only in binary format (Like Read only, but for a binary file instad of a text file.) Read only in binary format. File pointer starts at the beginning of the file.
r+ - Read/Write The file pointer starts at the beginning of the file.
w - Write only If the file exists, it is overwritten. The file pointer starts at the beginning of the file.
wb - Write only in binary format (Like Write only, but for a binary file instead of a text file.) If the file exists, it is overwritten. If the file does not exist, a new Read only file is created.
w+ - Read/Write If the file exists, it is overwritten. If the file does not exist, a Read/Write file is created. The file pointer starts at the beginning of the file.
wb+ - Read/Write in binary format (Like Read/Write, but for a binary file instead of a text file.) If the file exists, it is overwritten. If the file does not exist, a Read/Write binary file is created. The file pointer starts at the beginning of the file.
a - Append The file pointer starts at the end of the file. If the file does not exist, the file is created.
ab - Append in binary format (Like Append, but for a binary file instead of a text file.) The file pointer starts at the end of the file. If the file does not exist, the file is created.
a+ - Append/Read (Like Append, but for a read-only file instead of a text file.) The file pointer starts at the end of the file. If the file does not exist, the file is created.
ab+ - Append/read in binary format (Like Append, but for a read-only binary file instead of a text file.) The file pointer starts at the end of the file. If the file does not exist, the file is created.

Create a new directory by using the os module

import os
os.mkdir("test")
#Change directory using the same module
os.chdir("test")

Get the current working directory

os.getcwd()

Remove a directory:

os.rmdir("test")

Delete a file by using the os module

os.remove("filename.txt")

Open a file in Python

The following command opens a file in Read/Write binary format. After that, you will be able to access the attributes of the file.

file_var = open("filename.txt", "wb+")
#Returns the name of the file
print "File Name:", file_var.name

#Returns if the file is closed or not
print "Closed :", file_var.closed

#Returns the mode the file was opened with.
print "Opening Mode :", file_var.mode

#Returns false if space is explicitly required with print.
print "Softspace flag :", file_var.softspace

Close a file in Python

It is good practice to close files when you are done working with them.

file_var.close()

Determine whether a file exists

os.path.isfile("/path/to/file.txt")

Determine whether a directory exists

os.path.exists("/path/to/directory")

Put it all together

#!/usr/bin/env python2
#Import all of the necessary modules for our script.
import os

#Create a new directory for us to work in, if it is not there already.
if os.path.exists("test") == False:
  os.mkdir("test")

#Change directories to our new directory
os.chdir("test")

#Show us where we are now
print os.getcwd()

#Create our file "filename.txt"
file_var = open("filename.txt", "w+")

#Show all our file attributes
print "File Name:", file_var.name
print "Closed :", file_var.closed
print "Opening Mode :", file_var.mode
print "Softspace flag :", file_var.softspace

#Close our file.
file_var.close()

Write to a file

#!/usr/bin/env python2
#Open our file
file_var = open("filename.txt", "r+")

#write something simple to test our script
file_var.write("This is just a glimpse of things to come")

#close the file
file_var.close()

Read from a file

#!/usr/bin/env python2
#Remember use r when reading, w tends to overwrite files.

#Open the file
file_var = open("filename.txt", "r+")

#Save the data Python is reading to a variable.
data = file_var.read();

#Print out the data
print "Data: ", data

#Close our file
file_var.close()

Clean up

After you are done investigating the files that you have created, you can run the following command to clean up.

#!/usr/bin/env python2
#Import the necessary modules
import os

if os.path.isfile("test/filename.txt"):
  os.remove("test/filename.txt")
if os.path.exists("test")
  os.rmdir("test")
